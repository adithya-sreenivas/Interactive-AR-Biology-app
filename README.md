# Interactive-AR-Biology-app
Developed an interactive AR educational app using Unity, Vuforia, and C# that allows users to scan human organ cards to explore 3D models, watch educational videos, and attempt quizzes. The project features floating AR UI, real-time interaction, multimedia integration, and Android support for immersive learning.
An augmented reality educational app built using Unity and Vuforia.

## Features
- Scan organ cards
- Interactive 3D models
- Floating AR UI
- Educational videos
- Quiz system

## Tech Used
- Unity
- Vuforia
- C#
- Android Build Support

## Challenges Faced
- Video rendering issues in AR
- UI layering with world-space canvas
- Tracking stability
- Video playback synchronization

## What I Learned
- AR interaction systems
- Unity UI workflows
- VideoPlayer rendering modes
- Debugging complex issues
